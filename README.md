# Flex Platform Spotlight

Marketing website content and HTML for the **Flex Platform** — positions Flex as a commercial energy optimisation platform for businesses. Used for BD, partner presentations, and customer-facing comms.

## Published URLs

Served via GitHub Pages from the `main` branch root:

- **Blackcurrent version:** https://blackcurrent-energy.github.io/flex-platform-spotlight/flexspotlight_bc.html
- **Flex Energy version:** https://blackcurrent-energy.github.io/flex-platform-spotlight/flexspotlight_flex.html

Note: there is no `index.html`, so the bare root (https://blackcurrent-energy.github.io/flex-platform-spotlight/) 404s — link directly to one of the two pages above.

## Files

| File | Purpose |
|------|---------|
| `flexspotlight_bc.html` | Current live version — Blackcurrent brand |
| `flexspotlight_flex.html` | Current live version — Flex Energy brand (Farmlands co-brand) |

## Running locally

Open `flexspotlight_bc.html` or `flexspotlight_flex.html` directly in a browser. No build step.
