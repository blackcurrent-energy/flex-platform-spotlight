# CLAUDE.md — Flex Platform Spotlight (Marketing Website)

## Purpose

Marketing website content and HTML for the **Flex Platform** — positions Flex as a commercial energy optimisation platform for businesses. Used for BD, partner presentations, and customer-facing comms.

## Files

| File | Purpose |
|------|---------|
| `flexspotlight.html` | Current live version of the site |
| `flex_platform_content_brief_final.md` | Full content and design brief — source of truth for copy and positioning |
| `flex_platform_master_document.md` | Master content document |
| `flex_platform_feature_description.md` | Feature descriptions |
| `flex_website_copy_final.md` | Final approved copy |
| `flex_spotlight_reverse_brief.md` | Reverse brief |

## Key positioning (from content brief)

Core objective: Position Flex as a **commercial energy optimisation platform** that helps businesses reduce costs, improve performance, avoid risk, and maximise ROI.

Not a consumer product — audience is SMB, SME, industrial customers and channel partners.

## Running the site

Open `flexspotlight.html` directly in a browser. No build step.
